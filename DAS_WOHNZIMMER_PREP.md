# Das Wohnzimmer - Pre-Launch Preparation

## Visit Objectives

### Day 1: Discovery & Setup
1. **Meet with owner** - Understand their current workflow
2. **Identify POS system** - What Kassensystem do they use?
3. **Get WiFi access** - For testing on-site
4. **Document menu** - All drinks, prices, categories
5. **Set up venue in Supabase** - Real data entry

### Day 2: System Integration
1. **POS Integration research** - Can we connect to their Kassen?
2. **Create staff accounts** - Owner, manager, bartenders
3. **Barcode scanning** - Scan all inventory items
4. **Test check-in flow** - iOS app at the venue

### Day 3: Full Testing
1. **Points system** - Check-in → Points → Rewards
2. **Booking flow** - Make test reservations
3. **Inventory tracking** - Transfer, update, variance
4. **Task assignment** - Create and complete tasks

### Day 4: Staff Training
1. **Owner training** - Dashboard, reports, management
2. **Bartender training** - Tasks, basic functions
3. **Inventory manager** - Full inventory workflow

### Day 5: Soft Launch Prep
1. **Fix any issues** found during testing
2. **Final walkthrough** with owner
3. **Set launch date** - Internal then public
